# Finance Tracker - Premium Fintech App

A modern, high-end React Native (Expo) application for personal finance management. Optimized for a rich UI/UX, combining **PayU** aesthetics with **Fundly** glassmorphism.

## 🚀 Features

- **Rich Visuals**: Glassmorphic cards, mint-to-teal gradients, and Archivo-inspired typography.
- **Dynamic Money Flow**: Interactive bar charts for Weekly and Monthly spending analysis (Default: Weekly).
- **Core Tracking**: Add, edit, and categorize income and expenses with ease.
- **Budget Guard**: Set monthly limits and track progress with glassmorphic status bars.
- **Smart Insights**: Get carousel-driven alerts on your spending trends.
- **INR Support**: Native currency support (₹) for all transactions.
- **Dark & Light Mode**: Seamlessly transition between themes with optimal contrast.
- **Data Persistence**: Offline-first experience using `AsyncStorage`.

## 🛠️ Tech Stack

- **Framework**: Expo (Managed Workflow)
- **Navigation**: Expo Router (v3+)
- **Icons**: Lucide React Native
- **Animations**: React Native Reanimated & Moti
- **State Management**: Zustand (with Persistence)
- **Charts**: React Native Gifted Charts
- **UI Components**: Expo Blur, Expo Linear Gradient

## 📦 Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd FinanceTracker
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the application**:
   ```bash
   npx expo start
   ```

4. **Run on Device**:
   - Install the **Expo Go** app on your phone.
   - Scan the QR code from the terminal to launch the app.

## 🎨 Design Philosophy

The app is built with a "Fintech First" approach:
- **Depth**: Using blur intensities and translucent overlays to create a sense of hierarchy.
- **Clarity**: High-contrast typography and color-coded category icons for immediate recognition.
- **Micro-interactions**: Smooth transitions and haptic-like animations for every action.
